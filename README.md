# Name

「らくらくCoffee」・・・カフェのECサイト

# Features

[Next.js](https://nextjs.org/)、Type Scriptで作成したECサイトです🌟 
【実装内容】
・ログインページ
・新規ユーザ登録ページ
・一覧表示ページ
・商品詳細表示ページ
・カートページ
・注文確認・決済ページ
・決済完了ページ
